# dotfiles

Personal configuration files that may need to be backed up or synced across machines.

Planned config areas:

- `nvim/` - Neovim configuration
- `kitty/` - Kitty terminal configuration
- `rime/` - Rime input method configuration
- `macos/` - macOS defaults and setup helpers

Keep machine-specific secrets, tokens, and generated caches out of this repository.

## macOS setup

Speed up Dock auto-hide on a new Mac:

```sh
./macos/dock-fast-autohide.sh
```
